Show a user-friendly detailed map of hotels (and other accommodations, like a b&B or hostel) with a shortcode by using Google Maps and HotelRoomScanner.com.

== Description ==

Display a Google Map with the nearest accommodations. The hotels, B&B's, resorts and hostels will be loaded from the current data set of [HotelRoomScanner](https://hotelroomscanner.com).

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the photo gallery from the Plugins page.

Note: This plugin needs at least PHP 5.3+ to work as expected. PHP 5.2 is NOT supported. This plugin needs the PHP modules: cURL and filter.
